# LSP Billing Example (Python)

This repo demonstrates the Liskov Substitution Principle (LSP) using a billing + licenses example.

First, you will need to create your virtual environment:

$ virtualenv venv
$ source venv/bin/activate


## Run the dependencies

pip install -r requirements.txt


## Run the example

python main.py
